Detailed Description

Application headquarter.

It contains references to everything the application needs at runtime. Using a single structure to handle references is a choice made necessary by G* callback functions prototypes that allow only one parameter as data. This way accessing multiple application references is possible by a single callback (like updating multiple ui fields in response to an event).
