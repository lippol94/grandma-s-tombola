Detailed Description

Keeps references to Launcher widgets.

Launcher is the part of client application that drives the user through the pre-game part of execution flow. So everything from creating or joining a game, setting up username and game name to actually start it is handled by the Launcher. Obviously, it's the first UI part showed to the user when he launches the client.
